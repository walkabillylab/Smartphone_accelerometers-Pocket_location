Linear imputation, metadata, Counts,, and new features

In this markdown, we use the linear imputed data. metada, counts and new features variables are used for this model. We read the CSV file which contains x,y and z Counts and then apply the following models using the ApplyModels function from BEAP package:

  • RF, Random Forest
  • LDA, Linear Discriminant Analysis
  • NB, Naive Bayes
  • KNN, k-Nearest Neighbors
  • SVM, Support Vector Machines
  • DT, desicion Tree
