Linear imputation, X,Y,Z,metadata , and Counts

In this markdown, we use the linear imputed data. X, Y, Z-axis, metada, and counts variables are used for this model. We read the CSV file which contains x,y and z Counts and then apply the following models using the ApplyModels function from BEAP package:

  • RF, Random Forest
  • LDA, Linear Discriminant Analysis
  • NB, Naive Bayes
  • KNN, k-Nearest Neighbors
  • SVM, Support Vector Machines
  • DT, desicion Tree

# models canot work with strin

Note that we need to exclude Participants number 112 , 121 .122 , and 132 as they are not properly classified. Also, to save time, we run the models on 2 percent of the data. Check the code for more details.

Apply models

results <-
    Beap::ApplyModels(
        working_df = working_df,
        model_names = c("RF", "LDA", "NB", "SVM", "KNN", "DT"), #c("RF", "LDA", "NB", "SVM", "KNN", "DT")
        split_ratio = 0.7,
        shrink = 0.02,
        return_plots = TRUE,
        save_results = FALSE
    )
