Read the Steps.RMD for more details. To prepare the data go to the"important_code" folder and 

1- Read How_to_extract_Ethica.txt to extract Ethica from raw data
2- Run Combine_ethica_data.RMD to combine all Ethica data
3- Run merge_Ethica_Jaeger.R
4- Add metadata by running Add_metadata.RMD


The "Ethica_Data_Preparation_Steps_Explanation.RMD" file contains all the code and explanations.
